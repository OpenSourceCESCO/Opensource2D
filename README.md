## 규칙
회의 후 추가할 예정

### commit 관련

* 커밋 메시지 기호 통일   
사용하지 않아도 무방하나, 사용시에는 아래의 규칙을 따를 것
```
추가 : +
삭제 : -
수정 : ^
```
